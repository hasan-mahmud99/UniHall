Place hall images in this folder with the following filenames:
- ASH.jpg  (Basha Shaheed Abdus Salam Hall)
- AMU.jpg  (Bir Muktijuddha Abdul Malek Ukil Hall)
- HBK.jpg  (Hazrat Bibi Khadiza Hall)
- JSH.jpg  (July Shaheed Smriti Chhatri Hall)
- NFH.jpg  (Nabab Foyzunnessa Choudhurani Hall)

The app prefers local images under /halls first, then falls back to official URLs, then to generic images.
